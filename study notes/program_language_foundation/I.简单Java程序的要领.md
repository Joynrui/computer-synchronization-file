# Java笔记

## I.简单Java程序的要领

**1，Java程序有大小写的区分。**

**2.Java程序“方法”命名首字母要大写。**

## II.注意

java变量的声明与定义无区分。不同于C/C++，C/C++在函数声明与定义上，有明显区分。

**eg:   C/C++:   int i=10;         此为一个函数定义；**

​        **extern int i=10;           此为一个函数声明；**

**Java中不作区分。**

**eg:  final double SALARY=6000;** 

#:Java中的变量声明习惯上用**全大写**。
